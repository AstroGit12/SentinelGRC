# SentinelGRC

A production-grade, high-fidelity Governance, Risk & Compliance platform built with Next.js 14, TailwindCSS, and shadcn/ui.

## Features

- **Dashboard**: Command center with KPIs, heatmaps, and compliance velocity charts
- **Risk Register**: Comprehensive risk management with interactive data tables
- **Vendor Risk Assessment**: Multi-step security questionnaire with PDF certification
- **ISO 27001 Controls**: Knowledge base with implementation tracking
- **Cloud Simulator**: Interactive AWS security configuration testing
- **GDPR Tracker**: Data visualization of compliance fines and trends

## Tech Stack

- Next.js 14 (App Router)
- TailwindCSS + shadcn/ui
- Zustand (State Management)
- Recharts (Data Visualization)
- Framer Motion (Animations)
- jsPDF (PDF Export)
- Sonner (Notifications)

## Getting Started

Open https://sentinel-grc.vercel.app/ to view the application.

## Design Philosophy

This is an **enterprise internal tool**, not a marketing site. The design prioritizes:

- Data-dense interfaces
- High contrast for readability
- Trustworthy, professional aesthetics
- Minimalist but feature-rich

Inspired by Vanta, Splunk, and Linear.
